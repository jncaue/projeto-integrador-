Projeto integrador-

Integração da matéria de biologia, que uni conhecimentos com autoria web (HTML, CSS, Js e Sass);
Passo a passo para a construção de um ecossistema
Tema selecionado e trabalhado dentro de sala na matéria de biologia - Criação de um terrário auto-sustentavél
Página Biotech, estruturada por Juan Cauê Menezes da Silva - 20221064010034 e Luiz Gustavo da Silveira Targino - 20221064010005


    RETIRADO DO MODELO PRONTO:  "https://html5up.net/minimaxing" (minimaxing)
    IMAGENS RETIRADAS DE BANCOS DE IMAGENS E INTELIGÊNCIAS ARTIFICIAIS: pixabay e chatGPT


Estruturação de blocos dentro de uma wrapper:
Header - cabeçalho da página, contendo uma navbar e icone da página; 
Banner - apresentação do tema dentro de um h1;
Main   - apresentação de informações dentro de colunas;
    - row, main row - centraliza a estrutura

Footer - finalização da página;
    - ul - ícones dentro de uma anchor com link ligando a redes sociais

São todos modificados por uma main.css, main.js e uma extensão da linguagem de css


6 páginas ilustrativas, sendo 3 delas dentro de uma nav-bar e uma no footer

    nav-bar - "index.html", "nosso jardim.html", "galeria.html"
    buttons - "maisinformações.html", "vejadetalhadamente.html"
    footer  - "central de relacionamento.html"
