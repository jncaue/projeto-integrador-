Projeto integrador-

Integração da matéria de biologia, que uni conhecimentos com autoria web (HTML, CSS, Js e Sass);
Passo a passo para a construção de um ecossistema
Tema selecionado e trabalhado dentro de sala na matéria de biologia - Criação de um terrário auto-sustentavél
Página Biotech, estruturada por Juan Cauê Menezes da Silva e Luiz Gustavo Targino

Estruturação de blocos dentro de uma wrapper:
Header - cabeçalho da página, contendo uma navbar e icone da página; 
Banner - apresentação do tema dentro de um h1;
Main   - apresentação de informações dentro de colunas;
Footer - finalização da página;
São todos modificados por uma main.css, main.js e uma extensão da linguagem de css
